Publish to Web
Your edits are saved only in this browser. To make them appear on the public website (and for other people), publish them into a content.json file.
Step 1 — Download your content
Bundles everything you edited (experience, skills, photos, logos, theme, backgrounds) into one content.json file.
Step 2 — Put it in your project & push
Move the downloaded content.json into your project folder (next to index.html), then run: 
git add content.json
git commit -m "Update content"
git push
The website reads content.json on load, so visitors will see your real content.
Restore from a file (optional)
Load a content.json back into this browser — useful on a new computer or to recover your data.
