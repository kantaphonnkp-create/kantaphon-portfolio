# Kantaphon Aochampa — Portfolio

Personal portfolio website for **Kantaphon Aochampa** — Marketing & Business Development.

## Contents

| File | Description |
|------|-------------|
| `index.html` | The public portfolio site |
| `admin.html` | Built-in CMS to edit content (experience, competitions, certificates, portfolio, skills, theme colors, backgrounds, profile photos) — data is stored in the browser's `localStorage` |
| `PRODUCT.md` / `DESIGN.md` | Product brief & design system |

## Usage

Open `index.html` in a browser to view the portfolio.
Open `admin.html` to edit content — changes are saved locally in your browser and reflected on the site after a reload.

## Editable sections (via admin.html)

- Work Experience (with brand logos & "My Work" pieces)
- Competitions, Certificates, Portfolio items
- Skills (with multiple images per skill)
- Theme colors (live preview)
- Per-section background images with opacity
- Hero & About Me profile photos (independent)
